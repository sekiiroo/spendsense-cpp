# 💰 SpendSense - Personal Expense Tracker

A console-based Personal Expense Tracker developed in **C++** that helps users manage daily expenses efficiently. The application supports expense management, searching, editing, reporting, and persistent data storage using file handling.

---

## ✨ Features

- ➕ Add Expense
- 📋 View All Expenses
- 🔍 Search Expense by Date
- 📂 Search Expense by Category
- ✏️ Edit Existing Expense
- ❌ Delete Expense
- 📈 Monthly Expense Summary
- 💸 Highest Expense in a Month
- 📅 View Expenses Between Two Dates
- 💾 Automatic Save & Load using File Handling
- ✅ Date Validation
- ✅ Amount Validation
